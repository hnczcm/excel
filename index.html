插件概述

畅享星空AI小助手是一款功能强大的Excel插件，旨在帮助用户快速高效地完成各种Excel操作，包括复杂公式计算、数据格式化、图表生成，以及通过AI模型提供智能辅助。该插件通过简洁友好的功能区界面，为用户提供一站式Excel增强体验。

插件功能区界面展示

AI智能助手

AI聊天助手

功能介绍：直接在Excel中调用豆包、DeepSeek等先进大模型进行交流，解答各类问题并提供智能辅助。

使用方法：

1. 点击功能区中的"AI聊天"按钮

2. 在弹出的对话框中输入您的问题或需求

3. AI会分析您的问题并给出相应的回答

4. 您可以继续提问或将AI提供的解决方案应用到工作表中

应用场景：

解答Excel使用问题
数据分析咨询
编写公式帮助
自然语言查询
AI生成图表

功能介绍：利用AI分析数据并生成合适的图表，智能推荐最适合数据特性的可视化方式。

使用方法：

1. 选中需要可视化的数据区域

2. 点击"AI生成图表"按钮

3. AI会分析数据特性并推荐合适的图表类型

4. 选择您喜欢的图表方案，AI会自动生成并插入工作表

应用场景：

数据可视化
报表制作
演示准备
趋势分析
AI公式专家

功能介绍：智能识别数据需求，生成复杂Excel公式，节省编写复杂函数的时间。

使用方法：

1. 选择目标单元格

2. 点击"AI公式专家"按钮

3. 用自然语言描述您的计算需求（如"计算这些销售数据的季度增长率"）

4. AI会自动生成相应的Excel公式并应用到选定区域

应用场景：

复杂计算
条件统计
逻辑函数
数据处理自动化
万能神器

功能介绍：提供全面的AI辅助功能，解决各类Excel处理需求，一站式解决常见数据分析问题。

使用方法：

1. 点击"万能神器"按钮

2. 选择您需要的功能类型

3. 根据提示选择数据区域或输入具体需求

4. AI会执行相应操作并返回结果

应用场景：

数据清洗
格式转换
条件筛选
统计分析

注意： AI功能依赖于网络连接和相应的API服务。请确保您的计算机已连接互联网，并在配置中正确设置了API密钥。

数据处理工具

性别与打钩打×工具

功能介绍：快速在单元格中添加性别标识或勾选/叉号标记，方便数据整理和分类。

使用方法：

  1. 选择需要处理的单元格区域
  2. 点击"性别"或"打钩打×"按钮
  3. 在弹出的对话框中选择所需格式（男/女；勾/叉等）
  4. 完成操作后，相应标记会自动添加到选中单元格

号码格式化工具

功能介绍：自动格式化各类号码，如手机号、身份证号等，使数据更规范、易读。

功能 效果 适用场景
手机号格式化 将11位手机号转为 133-1234-5678 格式 客户信息表、联系人列表
身份证号格式化 将身份证号转为标准格式并验证有效性 人员信息管理、身份验证
银行卡号格式化 按每4位分组显示银行卡号 财务数据、支付信息

数值格式化工具

功能介绍：提供各种数值格式化选项，包括货币、千分位、小数位等，美化数据展示。

主要功能：

  • 元 - 将数值转换为人民币格式（如 ¥1,234.56）
  • 万 - 将数值转换为"万"为单位的格式（如 123.46万）
  • 逗号 - 添加千位分隔符（如 1,234,567）
  • 壹 - 将数字转换为中文大写格式（如 壹仟贰佰叁拾肆元伍角陆分）

四舍五入工具

功能介绍：提供多种舍入方式，包括四舍五入、向上舍入、向下舍入等，精确控制数值显示。

使用方法：

  1. 选择需要舍入处理的数值单元格
  2. 点击对应的舍入按钮（四舍五入、向上舍入、向下舍入等）
  3. 在弹出的对话框中设置保留小数位数
  4. 系统会按指定方式处理数值并更新单元格

文本处理工具

功能介绍：提供一系列文本处理功能，如大小写转换、自动换行、文本提取等，优化文本数据处理效率。

主要功能：

  • 首字母大写 - 将文本的首字母转为大写
  • 全部大写 - 将文本全部转为大写字母
  • 全部小写 - 将文本全部转为小写字母
  • 自动换行 - 为单元格设置自动换行属性
  • 文本筛选 - 保留或移除特定类型的字符（如中文、英文、数字等）
  • 提取信息 - 从复杂文本中提取特定信息（如姓名、日期等）

界面增强功能

行列高亮工具

功能介绍：增强Excel的可读性，通过高亮显示当前选中单元格所在的行和列，方便数据查看和编辑。

使用方法：

  1. 点击"行列高亮"按钮开启功能
  2. 选择任意单元格，插件会自动高亮显示该单元格所在的行和列
  3. 点击"聚光灯设置"可以调整高亮颜色和显示模式（仅行、仅列、行列同时）
  4. 点击"关闭高亮"可随时关闭此功能

高亮功能对大型表格或频繁数据查找非常有用，可以显著提高工作效率。

常见问题解答

Q: 如何获取API密钥？

A: 您可以访问相应AI服务提供商的官方网站（如豆包、DeepSeek等）申请API密钥，然后在插件的配置模块中进行设置。通常需要注册账号并按照平台指引创建应用获取密钥。

Q: 插件支持哪些Excel版本？

A: 本插件支持Excel 2016及以上版本，包括Office 365。较旧版本可能无法支持部分高级功能。

Q: 如何更新插件？

A: 插件会自动检查更新，您也可以在官方网站下载最新版本进行手动更新。更新后需要重启Excel以应用新版本。

Q: 使用AI功能是否需要联网？

A: 是的，AI功能需要连接到互联网以调用相应的语言模型服务。其他基础功能（如格式化、文本处理）可在离线状态下使用。

Q: 数据安全性如何保障？

A: 插件对通过AI服务处理的数据采取了严格的安全措施，不会永久存储用户数据。所有数据传输均采用加密方式，确保信息安全。

使用技巧

  1. 组合使用功能 - 例如，先使用AI生成数据，再应用格式化工具进行美化，最后使用行列高亮功能进行检查
  2. 利用快捷键 - 熟悉插件的快捷键可以大大提高操作效率，常用功能都设有快捷键
  3. 保存常用设置 - 对于经常使用的格式化选项，可以在配置中设为默认值，减少重复设置的时间
  4. 定期更新API密钥 - 确保AI功能正常运行，建议定期检查和更新API密钥的有效性
  5. 灵活运用AI提示 - 使用AI功能时，清晰具体的提示语可以获得更准确的结果
